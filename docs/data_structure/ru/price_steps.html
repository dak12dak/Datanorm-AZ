Столбец Назначение Тип данных Источник Пример
step_code Код ступени TEXT Поле 4 записи Z 01
description Описание ступени TEXT Поле 6 записи Z Staffelpr. 45 bis 899
price_kind Тип значения INTEGER Поле 8 записи Z 1
sign Знак надбавки/скидки TEXT Поле 9 записи Z +
base_price_type Базовая цена INTEGER Поле 10 записи Z 2
value Значение стоимости REAL Поле 12 записи Z 626.0
min_quantity Минимальное количество REAL Поле 15 записи Z 45.0
max_quantity Максимальное количество REAL Поле 16 записи Z 899.0

Примечания:

price_kind: 1 = ступенчатая цена, 2 = сумма надбавки/скидки, 3 = процент.

sign: + = надбавка, - = скидка.

base_price_type: 1 = прейскурантная цена, 2 = закупочная цена.

min_quantity и max_quantity определяют диапазон количества, для которого действует данная цена.

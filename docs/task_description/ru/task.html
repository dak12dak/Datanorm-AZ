Task Description (Russian)

Чтение следующего текстового файла:

  • Технические характеристики: текстовый файл может содержать до 2 миллионов строк
  • Чтение в фиктивную базу данных
  • Расчет продажной цены
  • Проверьте, доступен ли товар, используя номер товара

Тип записи А (статья)

  • Поле 3 = номер артикула
  • Поле 4 = название элемента
  • Поле 7 = 1 = значение прейскурантной цены, 2 = закупочная цена, 9 = прейскурантная цена
    • ⚠ Является ли цена в поле 9 прейскурантной ценой или ценой покупки?
  • Поле 9 = Цена

Тип тарифа Z (ступенчатая цена)

  • Поле 3 = номер артикула
  • Поле 8 = 1 = поэтапная цена, 2 = сумма надбавки/скидки, 3 = процентная надбавка/скидка
  • Поле 9 = + = надбавка, - = скидка
  • Поле 10 = 1 = прейскурантная цена, 2 = цена покупки
    • ⚠ Является ли цена/процент в поле 8 прейскурантной ценой или ценой покупки?

Структура таблицы статей

Столбцы: - Номер позиции - Название позиции - Прейскурантная цена - Скидка поставщика (в процентах) - Закупочная цена - Накладные расходы (в процентах) - Расчетная цена покупки - Наценка (в процентах) - Цена продажи
